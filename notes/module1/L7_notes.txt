What is Token Authentication?

- A security mechanism where a server issues a token (a string) to a client after successful authentication. The client then uses that token to make authorized requests.

Why use Token Authentication?

- Stateless: no need to store user session data on the server
- Commonly used in modern APIs, especially within single page apps (SPAs) and mobile apps
